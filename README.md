# Bifrost - AuthN & AuthZ


Bifrost is a minimal AuthN & AuthZ server written in Go. It is fast, reliable, secure. It supports OIDC, OAuth and SAML for applications using Bifrost as a IDP. It uses Aerospike for high throughput and PostgreSQL for reliable consistent storage

## Features 
 
- **Multi Authentications Frameworks**
    - SAML - Simple Assertion Markup Language - (XML Based) | Standard Industry Framework (Old)
    - OAuth - 
    - OIDC - OpenID Connect | Standard Industry Framework (New)
- **Low Latency**
    - Uses Aerospike for fast key value lookups for validations and session verification

- **Self hosted**
    - Add your app tokens to the secrets and start using bifrost as your own
